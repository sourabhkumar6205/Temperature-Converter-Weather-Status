# Temperature Converter & Weather Status (HTML + JavaScript)

This project is a simple Temperature Converter tool that converts temperature from Celsius to Fahrenheit and also displays weather status with an icon based on the entered temperature value.

## Features
- Convert Celsius to Fahrenheit instantly
- Shows weather condition like Very Cold, Cold, Normal, Hot, Very Hot
- Displays weather emoji icon for better visual feedback
- Clean and modern user interface
- Fully responsive and easy to use
- No frameworks required
